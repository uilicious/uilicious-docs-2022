# Select Dropdowns

### **Select a dropdown**

Here, we will use this "Number" dropdown list for illustration.

<details>

<summary>Number</summary>

One

Two

Three

</details>

Specify the `option`;

```javascript
 I.select("Two");
```

Or, specify the `list` and the `option`:

```javascript
I.select("Number", "Two");
```

This selects the option "Two".
